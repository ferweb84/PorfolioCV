PORFOLIO ADAPTABLE (Responsive) con Bootstrap 5 <hr>

Este proyecto creado es una página web adaptable a dispositivos de distintos tamaños (este tipo de sitio web se conoce en inglés como "responsive"). 
Un portfolio adaptable o responsivo se refiere a una página web que se diseña para ser visualizada correctamente en todos los dispositivos.

El propósito de esta página web es mostrar el portfolio de proyectos de desarrollador y la experiencia. Incluye una descripción breve de motivación, experiencia, proyectos, artículos publicados y formas de contacto.

### ¿Qué es un Portfolio adaptable?

Tecnologías <hr>

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

Además, se incluyeron Google Fonts para personalizar la fuente y Bootstrap icons para incorporar íconos como flechas y logos de redes sociales populares.

Español <hr>

El texto de la página web está escrito en español, al igual que las clases y atributos personalizados. Las clases relacionadas con Bootstrap se incluyeron en inglés.
